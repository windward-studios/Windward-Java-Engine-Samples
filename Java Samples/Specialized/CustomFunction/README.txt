Windward Custom Functions
Last updated: 2010-10-15

These files can be used to create a WindwardCustomFunctions.jar file containing your own custom functions, to be used with the Windward Java engine, version 10.0.0.0 or newer.

The IML is a project file for IntelliJ, a Java IDE.

Important notes:
	1. Compile using Java version 1.4.
	2. Do not change the version of the JAR creating using this code.
	3. 2. To use, replace the assembly of the same name where you installed the Java engine.
