Windward Custom Callbacks
Last updated: 2013-01-15

These files can be used to create a WindwardCustomCallback.jar file containing your own custom callbacks, which are called by the Windward Java engine during particular events, version 13.0.0.0 or newer is required.

The IML is a project file for IntelliJ, a Java IDE.

Important notes:
	1. Compile using Java version 1.4.
	2. Do not change the version of the JAR creating using this code.
	3. 2. To use, replace the assembly of the same name where you installed the Java engine.
